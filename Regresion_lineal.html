Por la parte de la regresin lineal, se calculan los dos coeficientes (beta 1 y beta 0) del modelo de regresin lineal simple. Se genera la ecuacin de la regresin lineal una vez ingresados ambos coeficientes.

Con el objetivo de representar de una manera ms visual estos datos y la ecuacin, se utiliza un grfico nombrando ambos ejes con el GDP y Nivel de felicidad. En dicha grfica se agrega la lnea de la ecuacin que se ajusta mejor a los datos.

En relacin a dichos datos y ecuacin se realizan diversos clculos para analizar su comportamiento.

In[9]:
import numpy
import matplotlib.pyplot as plt
import statistics as st

data['log10GDP']=numpy.log10(data['GDP'])
x=data['log10GDP']
y=data['Felicidad']

xmedia=st.mean(x)
ymedia=st.mean(y)

xi_xm=x-xmedia
yi_ym=y-ymedia
xi_xm_yi_ym=xi_xm*yi_ym
sxi_xm_yi_ym=sum(xi_xm_yi_ym)
xi_xm2=(xi_xm)**2
sxi_xm2=sum(xi_xm2)
B1=sxi_xm_yi_ym/sxi_xm2

B0=ymedia-(B1*xmedia)

print("B1=",B1)
print("B0=",B0)
y_pred=B1*x+B0

plt.scatter(data['log10GDP'],data['Felicidad'],alpha=0.75, label='Datos')
plt.xlabel('log10 GDP')
plt.ylabel('Nivel de felicidad')
plt.title('GDP vs Nivel de Felicidad')
plt.plot(x,y_pred,alpha=0.75, label='Lnea de regresin')
plt.grid(True)
plt.legend()
plt.show()

7. El Residual Standard Error es una mtrica til que estima la desviacin estndar del error, es decir, el tamao del error que se encuentra al momento de realizar predicciones al utilizar el modelo. Cuando se tiene un valor pequeo esto indica que las predicciones son muy similares a los valores reales.

Por ltimo, para calcular la R^2, valor que indica el comportamiento del modelo, primero se utiliza el Total Sum of Squares (TSS) siendo la medida de variacin con respecto a la media. Teniendo este valor, y en conjunto conel RSS se obtiene el valor de la R^2.
